## Design Notes

### Crowd Level Indicators
- 🟢 Quiet (0-20 people)
- 🟡 Moderate (21-50 people)  
- 🔴 Busy (50+ people)
- ⚫ Data unavailable

### Temple Type Indicators
- 🌟 GPS Navigation available
- 🏛️ Information only
- ⭐ User favorites
- 🎯 Featured/recommended

### Performance Considerations
- Lazy loading for temple cards
- Image optimization with placeholders
- Progressive enhancement for real-time data
- Offline fallbacks for all content sections
